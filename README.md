# Windows-

## Python Virtual Environment Setup (Windows)

This project uses a Python virtual environment (`venv`).  
Follow the steps below to create and activate it on Windows.

---

### 1. Create Virtual Environment

Run this **once** inside the project root directory:

```powershell
python -m venv venv
```
```
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
```
```
venv\Scripts\Activate.ps1
```
